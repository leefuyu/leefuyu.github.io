JS一些基础知识点

发表于2020-10-03|更新于2023-04-03|JavaScript
|字数总计:2.7k|阅读时长:8分钟|阅读量:

js基础知识点

  • JavaScript的组成
    • ECMAScript — JavaScript语言基础
    • Web Apis
      • DOM - 页面文档对象模型
        • 大多数情况下都是在操作DOM
      • BOM - 浏览器对象模型
        • 比如一些前进 后退 ， 视口宽度
  • 浏览器本身自带的弹框
    • confirm 确认框
      • 进阶中 返回 true 和 false
    • prompt 输入框
    • document.write 在网页中写入内容
    • console.log 经常用于打印，用于调式代码
  • 变量
    • var
      • 是ES5提出的，声明变量 会有变量提升的情况，它是先使用，在声明，会将声明的变量提升到当前作用的最前面
      • 声明的变量可以重复赋值
    • let
      • 能够只声明变量，再进行赋值，同一个变量不允许重新赋值，没有变量提升的情况
      • 只声明变量不赋值的话，会返回undefined
    • const
      • 是常量，定义的变量，不能进行修改了，并且同一变量不能重复赋值，
      • const定义变量的时候,必须定义数据，否则会报错
      • 不需要变化的值才用变量，需要变化的值用变量

数据类型

  • 基本数据类型
    • number
      • NaN
    • string
    • undefined
    • null

  • 复杂数据类型

    • array
    • Object
    • function

  • 转义字符

    • 一般使用在 我们的字符串中，有多种符号，可以放我们的开发更方便
  • 返回false的几种方式
    • 6种情况
    • 0，””, undefined, null, false, NaN

一元运算符

  • 只有一个操作数的运算符 比如：num++
  • 自增运算符 (++)
    • num++
    • num—
      • 不同点
        • num++ 表示先返回值，再加1
        • ++num 表示先加1，后返回值-
  • 自减运算符（—）
    • 原理 同上

比较运算符

  • 在开发中比较常用 < > + - == === 等
  • 注意点
    • NAN 不等于任何值，包括它自己

逻辑运算符

  • 开发中常用
  • 列如
    • && 与 - 并且
      • 一假则假 - 两边都满足为true
      • 返回的值本身
    • || 与 - 或者
      • 一真为真
      • 两者满足一个即可
    • ！非- 取反
      • 真变假 假变真
  • 逻辑断路
    • 存在于&&和 | | 中，当满足一定条件会让右边的代码不执行
  • 运算符优先级
    • 在运算中 （） 优先级更高
    • 在开发中写代码的时候，需要先算的 直接使用 () 增加优先级

三大流程控制语句

  • 单分支，if…else — 逻辑判断- 符合条件的情况 进入if， 不满足进入else
  • 双分支，if…else if…else —— 不满足条件1 查看是否满足条件2，都不满足就不进入if判断

三元运算符

  • let result = 判断条件 ？ 代码1 ： 代码2
    • 先判断条件是否成立
      • 如果条件成立了，此时执行代码1，并将代码结果赋值给 变量
      • 如果条件不成立，此时执行代码2，并将代码的结果赋值给 变量

waitch语句

  • 用于处理具体的判断

    • 如果表达式的值为1，就进入语句1
      • 执行完之后退出

  • while循环（工作中几乎不用）

    • 先看可不可以执行，条件为true的情况下，执行循环体
      • 在执行的过程中，一定要有结束，否则的话就是死循环

  • for循环（工作中常用）

    • 一般情况下 循环中执行的代码 都是需要处理的数据
      • 比较适合明确循环范围次数的情况下使用
    • 在循环中 遇到 break 会立刻结束整个循环，然后执行循环之外的代码
    • 如果 遇到了 continue 会立刻结束本次循环，继续执行下一次循环

数组 Array

  • 本质上就是数据的集合，可以对数组进行增删改查操作
  • 数组中每一个值都有下标， 下标是从0开始的
    • arr.length可以获取当前数组的长度
  • 添加
    • 通过下标添加
      • 数组名[下标] = ‘新数据’
        • 当下标在数组中不存在，此时会新增这一项
        • 每次需要计算最后一项再往后一项的下标，计算麻烦
      • 长度添加
        • arr[arr.length] = ‘新数据’
          • 随着数组长度的变长， length也在变化，永远比最大的下标多1
      • 静态方法添加
        • arr.push(‘数据’) ———- 后添加
        • arr.unshift(‘ 数据 ‘) ———- 前添加
  • 删除
    • arr.pop() —- 在数组的最后删除一项，并且返回删除的这一项
    • arr.shift() —- 在数组的前面删除一项，并且返回删除的这一项
  • arr.splice()
    • splice(‘下标’， 删除的个数，添加1， 添加2)

函数

  • 函数可以吧具有相同或者显示的逻辑代码包裹起来，通过函数调用执行这些被包裹的代码， 提高代码的复用性，减少重复的代码。 —- 可以一次声明，多次调用
  • 参数
    • 函数的参数为 形参和实参
      • 形参 是在函数声明的，设置的参数，占位置
      • 实参 是在函数调用的时候，传入的参数，形参有几个就传入几个
    • 参数的默认值
      • 在我们的定义函数的形参时，可以设置默认值，在函数调用时，不传入实参，就不会报错，它会以默认值执行逻辑
  • 函数的返回值 - return
    • 我们在调用函数的时候，不需要知道函数的内容，只需要一个结果即可
    • 在函数内部，需要将结果return 出来
    • return表示函数返回的结果，函数已经结束，return之后的代码就不会执行了
  • 函数的三要素
    • 函数名
    • 函数的参数
    • 函数的返回值
  • 注意点
    • 函数名不能重名，卸载后面的函数会覆盖前面的
    • 预解析：
      • 会提升到浏览器的前面

作用域

  • 作用域分类
    • 局部作用域
      • 函数内部的区域
    • 全局作用域
      • 在script标签内，函数外部的区域
    • 块级作用域
      • let配合 { } 包裹起来的区域
  • 变量分类
    • 全局变量
      • 在acript标签内，函数外部声明的变量
      • 在任何地方都可以使用
    • 局部变量
      • 在函数内部声明的变量
      • 只能在函数内部使用
    • 局部变量
      • 在 { } 块级作用域中声明的变量，叫做 块级变量
      • 块级变量，只能在 当前 { } 内部使用

定义函数的方式

  • 函数声明 function fn() {}

  • 函数表达式 let fn = function() {}

  • 区别

    • 普通声明：可以先调用，后声明
      • 因为有预解析的存在，js代码执行之前会把所有的声明提升到最前
    • 函数表达式： 必须先声明赋值，再调用才行 - 如果声明放在前，函数表达式会报错

  • 匿名函数

    • 顾名思义 就是 就是没有名字的函数

      • 函数可以 自调用，会立即执行 —- 又叫 立即执行函数

      • 但是会报错，需要用 ( ) 把整个函数包裹起来

      • 在函数必须前 放 ;

      • 立即执行函数的应用

        • 在多人同时写代码的时，如果使用的是全局变量，很容易与其他人的全局变量互相影响， 这叫做 全局变量污染

        • 此时可以使用 立即执行函数,每个人的代码在单独的作用域中，不会相互影响

        • ;(function () {
    let a = 10
    console.log(a)
    function fn() {
        console.log('aaaa');
    }
    fn()
})()

// 小王写的代码，单独运行没毛病
;(function () {
    let a = 22
    console.log(a)
    function fn() {
        console.log('bbbb');
    }
    fn()
})()
//---------------------但是如果不是立即执行函数，全局变量就会互相影响了（全局变量污染）

对象

  • 对象是一个 无序的键值对的集合，可以用于描述生活中对象的特征与行为

  • 对象中的 静态特征 为 对象的属性

  • 对象中的动态行为（函数）叫做对象的方法

    • 使用 .语法 可以访问到对象中的属性和方法

  • 对象方法的取值

    • 点语法 - 可以直接取对象的属性和方法

    • 中括号语法 - 也叫做 关联数组语法 类似于把对象看做是数组取值

    • 点语法和中括号语法的区别

      • 点语法： 对象名.属性名 —- 简洁 —- 只能写属性名 —- 不支持变量
      • 中括号语法： 对象名[‘ 属性名 ‘] —- 灵活 —- 可以写字符串或者变量

  • 对象的增删改查

    • 查询对象
      • 对象.属性 && 对象[‘ 属性 ‘]
      • 对象.方法()
    • 重新赋值
      • 对象.属性 = 值
      • 对象.方法 = function() {}
    • 添加新的数据
      • 对象.新属性名 = 新值
    • 删除对象中的属性
      • delete 对象名.属性名

  • 对象的遍历

    • 遍历对象：一个一个的访问对象的全部属性

      • 对象没有数组的length长度，所以无法确定长度
      • 对象里面是无序的键值对，没有规律

    • 语法：

      • for (let key in obj) {}
        • 能够拿到对象中的所有key

      • Object.keys()

        • 会拿到所有key，并且 返回一个数组

      • Object.values()

        • 能够拿到所有的值(value), 并且返回一个数组
  • 内置对象
    • js的内置对象，是JavaScript自带的一些对象
    • 常见的内置对象中 有 Math， String， Array， Date等
    • 需要使用时，不需要调用 直接使用即可
      • 比如 Math对象
        • Math.PI: 生成圆周率 3.1415926
        • random：生成0-1之间的随机数（包含0不包括1）
        • ceil：向上取整
        • floor：向下取整
        • max：找最大数
        • min：找最小数
        • pow：幂运算
        • abs：绝对值
        • …
          • 绝对值 就是 只显示 正整数 不管是负数还是整数
      • 如果只想取整数的话
        • 使用 parseInt 配合 Math对象

数据类型

  • 值类型和引用类型
    • 值类型 - 简单数据类型
      • number string boolean undefined null
      • 存储值的时候，存储的是值本身，
      • 在内存角度来说 简单数据类型存储在 栈内存中
    • 引用类型 - 复杂数据类型
      • Array Object Function
      • 在存储的时候存储的 内存地址
      • 内存角度来说的话 存储在 堆内存中
    • 赋值特性
      • 简单数据类型 存储的是值本身，所以赋值的时候，就是值本身
      • 复杂数据类型 存储的是内存地址， 赋值的时候， 是内存地址
